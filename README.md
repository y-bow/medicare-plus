# MediCare Plus | Premium Healthcare Services

**MediCare Plus** is a high-end, boutique-style healthcare website designed to provide a luxurious and trustworthy digital experience for patients. Moving away from generic medical templates, this project focuses on an "editorial" aesthetic, combining sophisticated typography with modern web design principles to reflect excellence in medical care.

**[View the Live Website](https://mantramedicare.netlify.app/)**

---

### Technical Excellence
* **Responsive Design**: Fully optimized for mobile, tablet, and desktop views using CSS Flexbox and Grid.
* **Interactive Animations**: Implemented smooth "fade-in-up" entrance effects using the `IntersectionObserver` API for a seamless user experience.
* **Dynamic Navigation**: A custom mobile-responsive menu for easy navigation across all devices.
* **Form Validation**: Robust JavaScript-based validation on the contact/appointment form to ensure high-quality lead capture.

### AI-Powered Patient Support
* **Tawk.to Intelligent Chatbot**: Integrated a real-time live chat solution powered by **Tawk.to**.
* **AI Assistance**: The chatbot utilizes AI-assisted automation to handle common patient inquiries, providing instant responses and improving accessibility 24/7.

## Tech Stack

* **Frontend**: HTML5, CSS3 (Custom Properties, Grid, Flexbox)
* **Interactivity**: JavaScript (ES6+)
* **Chatbot**: Tawk.to (AI-enhanced)
* **Deployment**: Netlify
* **Version Control**: Git & GitHub

   ```bash
   cd medicare-plus
   ```
3. **Open in browser:**
   Simply open `index.html` in any modern web browser.
